README  

1. Para compilar usamos o Notepad++
   Assim, acessamos o código usando o CMD na pasta salva e compilamos com este código:
   > gcc helloworld.c -o helloworld
   
2. Para executar o programa deve ser executado no CMD em sequência da compilação com o arquivo .exe:
   > helloworld.exe
   
3. A saída do terminal deverá aparecer a frase: "Hello World", dessa forma a execução do programa foi um sucesso
